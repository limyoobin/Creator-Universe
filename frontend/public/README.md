# Public Assets

프론트엔드에서 정적으로 제공하는 이미지와 브랜드 파일입니다.

## 브랜드 파일

| 파일 | 용도 |
| --- | --- |
| `logo.png` | 웹사이트 로고, Apple touch icon |
| `favicon.png` | 브라우저 탭 아이콘 |
| `favicon.svg` | SVG 파비콘 백업 |
| `og-image.svg` | 링크 공유 미리보기 이미지 |
