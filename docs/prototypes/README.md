# Prototype Archive

초기 기획과 화면 실험에 사용했던 파일들을 보관하는 폴더입니다.  
현재 배포되는 서비스 코드는 루트가 아니라 `frontend/`, `src/`, `prisma/` 폴더를 기준으로 동작합니다.

## 보관 파일

| 파일 | 설명 |
| --- | --- |
| `creator-universe-homepage.html` | 초기에 만든 단일 HTML 홈페이지 데모 |
| `matching-dashboard.tsx` | 매칭 대시보드 UI 초안 |
| `smart-settlement-dashboard.tsx` | 스마트 정산 대시보드 UI 초안 |
| `multimedia-viewer.tsx` | 멀티미디어 뷰어 UI 초안 |
| `preview_server.py` | 단일 HTML 데모를 미리보기하던 Python 서버 |

## 주의

이 파일들은 참고용 아카이브입니다. 최신 UI와 백엔드 연동은 `frontend/src/App.tsx`, `frontend/src/styles.css`, `src/` 폴더를 기준으로 확인하세요.
